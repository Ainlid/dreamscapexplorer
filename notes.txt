importing meshes:
1. export .gltf
2. select built-in materials in Godot import settings
3. reimport and delete material files

collisions:
1. model collisions as separate object
2. generate trimesh static body from the collision object
3. hide the collision object

normal maps:
1. desaturate
2. 2px gaussian blur
3. height to normal
make sure repeat mode is set to true
